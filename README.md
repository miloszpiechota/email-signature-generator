

## Jak uruchomic projekt od zera

1. Otwórz terminal i przejdź do folderu frontendu -> cd frontend
2. Sprawdź, czy masz zainstalowane Node.js: -> node -v & npm -v
3. Jeśli nie, pobierz instalator ze strony: https://nodejs.org/en/download
4. Zainstaluj zależności. To polecenie pobierze wszystkie wymagane pakiety
 i utworzy katalog node_modules. -> npm install
5. Uruchom aplikacje -> npm run dev
6. Otwórz przeglądarkę pod adresem wyświetlonym w konsoli

7. W razie gdyby wystąpił w konsoli problem z brakiem wymaganej biblioteki przydatne może być -> npm install <nazwa biblioteki w consoli>
